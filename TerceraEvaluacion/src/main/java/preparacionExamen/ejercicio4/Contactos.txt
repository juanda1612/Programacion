Contactos
Crea un programa en Java para gestionar una lista de contactos. Cada contacto estará representado por la clase Contacto, que tendrá los siguientes atributos:

nombre: una cadena de caracteres que representa el nombre del contacto.
telefono: una cadena de caracteres que representa el número de teléfono del contacto.

Debes crear una clase llamada Gestor, que contendrá los siguientes métodos:

guardarContactos: Este método recibirá una lista de contactos y un nombre de archivo, y guardará los contactos en esa lista en el fichero.

imprimirContactos: Este método recibirá el nombre de archivo y mostrara el fichero.

Crea una clase Main donde deberás instanciar algunos contactos, añadirlos a una lista, utilizar el método guardarContactos de Gestor para guardar la lista en un archivo y luego utilizar el método imprimirContactos para mostrarlos por consola.
